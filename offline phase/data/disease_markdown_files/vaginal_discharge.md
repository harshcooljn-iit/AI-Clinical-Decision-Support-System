# Figure 1. NACO algorithm for the management of vaginal discharge

```mermaid
graph TD
    A[Vaginal discharge complaint] --> B{Partner with urethral discharge or genital sore?}
    B -- Yes --> C[Speculum examination]
    B -- No --> D[Speculum examination]

    C --> C1[Cervical mucopus]
    C --> C2[Curd-like vaginal discharge]
    C --> C3[Other abnormal vaginal discharge]
    C --> C4[No abnormal discharge]

    C1 --> C1a[Rx for gonorrhoea and chlamydia]
    C2 --> C2a[Rx for gonorrhoea, chlamydia and candidiasis]
    C3 --> C3a[Rx for gonorrhoea, chlamydia, trichomoniasis and bacterial vaginosis]
    C4 --> C4a[Rx for gonorrhoea and chlamydia]

    D --> D1[Cervical mucopus]
    D --> D2[Curd-like vaginal discharge]
    D --> D3[Other abnormal vaginal discharge]
    D --> D4[No abnormal discharge]

    D1 --> D1a[Rx for gonorrhoea and chlamydia]
    D2 --> D2a[Rx for candidiasis]
    D3 --> D3a[Rx for trichomoniasis and bacterial vaginosis]
    D4 --> D4a[No treatment]
```
