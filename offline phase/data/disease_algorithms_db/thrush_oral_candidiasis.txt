```text
DISEASE: Thrush Oral Candidiasis

ABBREVIATIONS:
- mg: milligrams
- kg: kilograms
- tab: tablet
- AIDS: Acquired Immunodeficiency Syndrome

DIAGNOSTICS:
- Thick white patches on an erythematous base in the oral mucosa.
- Plaques can spread to lips, buccal mucosa, tongue, and palate.
- Punctate areas of bleeding seen on the undersurface when plaques are removed.
- Symptoms may include pain, discomfort, anorexia, decreased feeding.

TREATMENT ALGORITHM:

IF Patient has Oral Candidiasis (Thrush) THEN

    // Non-pharmacological treatment
    Correct faulty sterilization technique of feeding bottle.
    Advise avoiding bottle feeding if possible.

    // Pharmacological treatment

    // First-line topical treatments
    IF Patient is suitable for topical treatment THEN
        IF Using Clotrimazole THEN
            Administer Clotrimazole 1% cream, gel, or lotion orally 3-4 times/day after feeding for 5-7 days (or 1-2 days beyond recovery).
        ELSE IF Using Miconazole gel THEN
            Administer Miconazole gel 25 mg orally 4 times/day for 5-7 days.
        ELSE IF Using Gentian violet THEN
            Administer Gentian violet 1% aqueous solution orally 1-2 times/day for 5-7 days.
            NOTE: Gentian violet can stain tissues and clothes.
        ENDIF
    ENDIF

    // Second-line systemic treatments (for resistant/chronic cases or major underlying disease)
    ELSE IF Patient has resistant/chronic thrush OR major underlying disease THEN
        IF Using Fluconazole THEN
            Administer Tab. fluconazole 3-6 mg/kg once daily for 5-7 days.
        ELSE IF Using Ketoconazole THEN
            Administer Tab. ketoconazole 3-6 mg/kg once daily for 5-7 days.
        ENDIF
    ENDIF

    // Investigation for resistant recurrent/chronic thrush without obvious predisposing factors
    ELSE IF Patient has resistant recurrent/chronic thrush AND no obvious predisposing factor/source of infection THEN
        Look for underlying endocrinopathy, immune disorder, AIDS, and malnutrition.
    ENDIF

ENDIF

// Patient Education
Emphasize bottle hygiene.
Advise care/hygiene of the nipple.
Advise treatment of vaginal candidiasis in the expectant mother if applicable.

CONTRAINDICATIONS:
- Gentian violet: Use with caution due to staining potential.
- Fluconazole/Ketoconazole: Use with caution in patients with major underlying disease, consider potential drug interactions and side effects. Specific contraindications depend on the individual patient's medical history and other medications.
```
